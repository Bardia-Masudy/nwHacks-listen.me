## Listen.me
A specialized speech assistant for people with Anomic Aphasia (Anomia), meant to silently help you find the words you're looking for, mid-conversation.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
